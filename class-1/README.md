# 自行操作事項

---

1. 安裝Git([Windows](https://git-scm.com/install/windows)、[Mac](https://git-scm.com/install/mac))、[SourceTree](https://www.sourcetreeapp.com/)

   如果你是Mac，看不懂Git的官方說明，可參考[這份文件](git-install-for-mac.md)

2. 申請[GitHub](https://github.com/)帳號
3. 用Intellij IDEA建立專案
4. Commit專案
5. 在GitHub建立Repository
6. 設定Remote
7. Push上去
